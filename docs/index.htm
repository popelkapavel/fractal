Fractal

Download: (32bit exe) fractal015.zip sources 

Fractal is simple tool for zooming fractals.

Use mandelbrot, find place, use 'S' key for speedup then hold 'J' key for julia and move mouse.
Finaly add antialias with 'A' key.

Change colors with '0' for random palette.

Source in C and assembly, makefile for gcc and nasm.

Mouse click:
left: 2x zoom in 
drag: zoom rectangle (ctrl => right to left or bottom to top do flip, shift withou aspect)
right: 2x zoom out (pan)
wheel: zoom, ctrl or shift move,alt rotate around cursor

Keys:
F1:  this help
Esc,q: exit
F11: full screen (or space, ctrl hides/shows menu)
m:   menu on/off
arrow: move by pixels (ctrl+shift rotate)
PgUp: move by page (PgDown,Delete,End)
Ins: zoom in (or Enter,Home/Backspace/Num+ zoom out)

r:   reset zoom (or Num-, shift,ctrl also set mandelbrot set)
j:   julia (shift juliab, ctrl mandelbrot)
h:   julia 3 (--||--) 
k:   julia 4 (--||--) 
n:   julia 16 (--||--) 
b:   julia 64 (--||--) 

1-9: color and palettes
~:   rotate rgb (shift reverse palette,ctrl rgb<=>cmy,ctrl+shift invert)
i:   invert colors,(shift reverse palette)
d:   double pallete (shift with mirror,ctrl triple)
c:   contrast (ctrl rgb independent, shift full palette)
t:   palette modulo on/off
0:   random palette (or Num0, ctrl invert,alt reverse

<>:  decrease/increase depth 2x (numpad /*)
[]:  rotate palette (shift 4x,ctrl 8x)
.,:  decrease/increase depth calculation
a:   2x2 antialias on/off (ctrl for 3x3,ctrl+shift 4x4)
s:   4x4 speedup on/off (ctrl for 8x8 nad shift for 16x16)
u:   flip vertical (o flip horizontal)

f:   save file with dialog (or F2, shift for 2x image size,ctrl 4x ctrl+shift 8x)
p:   write out###.gif file (ext from save dialog filter)

Command line:
-f fullscreen (F11)
-t topmost (F10)
-a antialias 2x2 (-aa 3x3)
-s speedup 4x4 (-ss 8x8,-sss 16x16)
-eb default file filter extension (-eb bmp, -eg gif, -ep pgm,-ea all)
-c# use # cores (default 2)


It is free for any use.
